Test page to check ViewImage PNG rendering with different image formats of IANA Registered Media Types, plus BMP format.

Prerequisites

  • copy /yacy/test/viewImageTest/test folder in /yacy/htroot/env/grafics
  • run a YaCy instance on localhost and 8090 port
